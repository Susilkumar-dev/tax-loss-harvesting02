# KoinX Tax Loss Harvesting Tool

A responsive React-based Tax Loss Harvesting tool that helps users optimize their tax liability by selecting holdings to harvest losses.

## Features

- 📊 Display capital gains (pre and post harvesting)
- 📋 Render holdings table with real API data
- ✅ Select/deselect holdings to see real-time impact on tax liability
- 💰 Calculate tax savings when harvesting reduces capital gains
- 📱 Fully responsive design for mobile and desktop
- ⚡ Loading states and error handling
- 🔄 "View All" functionality for holdings table

## Tech Stack

- React 18
- Tailwind CSS for styling
- Mock API services with simulated delays
